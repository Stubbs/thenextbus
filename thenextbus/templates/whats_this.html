The Next Bus - Near Me

If you phone has built in GPS and your browser supports the new location API you can now select the bus stop according to which one is nearest to you!

Press the Near Me button and TheNextBus will try and locate all bus stops within 500 meters of your current location.

If one of the stops doesn't have the reference number available, you won't be able to select it. Why not visit OpenStreepMap.org and add the data yourself?
